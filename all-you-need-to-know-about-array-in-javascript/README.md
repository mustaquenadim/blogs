# ALL YOU NEED TO KNOW ABOUT ARRAY IN JAVASCRIPT

## What is an array?

Simply, Array is a collection of data which can be stored in a variable.

```
let arr = [10, 20, 30, 40, 50];
```

## Initialization

Array can be initialized in 2 ways in JavaScript.

### Method 1

```
let arr = new Array;
```

### Method 2

```
let arr = [];
```
